
### Dependencies
- [Docker](https://docs.docker.com/get-docker/)
- Ruby

### Version
- Ruby 2.6.10-alpine

### Tests
To run the unit tests, run the commands below:

###### 1 - Build the Docker image:
```bash
$ docker-compose build
```
###### 2 - After, just run
```bash
$ docker-compose run challenge-rd
```
